---
title: Rate Limits
description: API rate limits per endpoint category.
---

| Endpoint category | Limit |
|-------------------|-------|
| Pull data | 100 req/min per API key |
| Adopt / Dispute | 200 req/min per API key |
| Verticals (public) | 60 req/min |

Exceeded limits return `429 Too Many Requests`.
