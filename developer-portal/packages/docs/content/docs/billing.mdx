## How pricing works

1. **Subscribe** to a data vertical
2. **Receive** delivery items as they are annotated
3. **Adopt** items you want → charged `unit_price_usd` per item
4. **Dispute** items you reject → no charge, item re-reviewed
5. Balance is deducted from your prepaid account

## Balance types

| Type | Description |
|------|-------------|
| **Available** | Funds ready to spend |
| **Frozen** | Reserved for pending deliveries |
| **Earnings** | Revenue from contributed data (future) |

## Top up

Add funds via Stripe Checkout in the dashboard (Billing tab). Quick amounts: $100, $500, $1,000, $5,000.

All transactions are recorded in the Billing → Transactions tab.
